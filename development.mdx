## Getting started

Before you begin, ensure you have:

- [<u>Yarn</u>](https://yarnpkg.com/getting-started/install) installed
- [<u>Docker</u>](https://docs.docker.com/get-docker/) installed and running

## Running the project locally

Follow these steps to set up and run your project locally:

**Step 1**: Install dependencies:

<CodeGroup>

``` yarn
yarn install
```

</CodeGroup>

**Step 2**: Start the project:

```bash
yarn run start
```

Congratulations, your project is now running locally\! Access the administration UI at [<u>http://localhost:1480</u>](http://localhost:1480).

It will also launch the following components:

| Container        | Port | Url                                                   |
| :--------------- | :--- | :---------------------------------------------------- |
| Contember Engine | 1481 | [<u>http://localhost:1481</u>](http://localhost:1481) |
| Postgres         | 1482 | [<u>http://localhost:1482</u>](http://localhost:1482) |
| S3               | 1483 | [<u>http://localhost:1483</u>](http://localhost:1483) |
| MailPit          | 1484 | [<u>http://localhost:1484</u>](http://localhost:1484) |
| Adminer          | 1485 | [<u>http://localhost:1485</u>](http://localhost:1485) |
| S3 Dashboard     | 1486 | [<u>http://localhost:1486</u>](http://localhost:1486) |

## Project overview

The project consists of the following packages:

#### [@app/admin](https://github.com/ascm00/fuglio-marketplace/blob/main/admin)

- **Location:** `./admin`
- **Description:** A React SPA that serves as the administration UI, built with Contember DataBinding for seamless integration with the Contember API.
- **Sentry:** [<u>Sentry</u>](https://sentry.io/) is integrated into the project. To enable Sentry, set the `VITE_SENTRY_DSN` environment variable in the `.env.production` file.

#### [@app/api](https://github.com/ascm00/fuglio-marketplace/blob/main/api)

- **Location:** `./api`
- **Description:** Defines the data model and permissions for the Contember API.

#### [@app/client](https://github.com/ascm00/fuglio-marketplace/blob/main/client)

- **Location:** `./client`
- **Description:** A type-safe GraphQL client for interacting with the Contember API, complete with generated types.
- **Documentation:** [<u>Client Documentation</u>](https://github.com/ascm00/fuglio-marketplace/blob/main/client/README.md)

## Deployment

To deploy the application, follow the steps outlined in the deployment script or documentation. Make sure all environment variables are properly set and that the target environment meets the required system dependencies.

**Step 1**: Make sure that you tested every new feauture properly on your local server.

**Step 2**: Make sure that you are in the main branch.

**Step 3**: Run command:

```bash
git push origin main:deploy/prod
```
